# Parag Data Analyst Portfolio

Static GitHub Pages portfolio for showcasing data analyst projects, business analytics skills, and dashboard-style case studies.

## Files

- `index.html` contains the portfolio content and project case studies.
- `styles.css` contains the responsive dashboard-style design.
- `script.js` powers project filtering, metric counters, and active navigation states.

## Customize

Replace these placeholders before publishing:

- `https://github.com/your-username`
- `https://www.linkedin.com/in/your-profile`
- `mailto:your.email@example.com`
- `resume.pdf`
- Project `GitHub placeholder`, `Dashboard placeholder`, and `Case study placeholder` links

## Publish On GitHub Pages

1. Create a GitHub repository for the portfolio.
2. Upload these files to the repository root.
3. In GitHub, go to Settings > Pages.
4. Set the source to the main branch and root folder.
5. Save, then open the GitHub Pages URL after it finishes deploying.
